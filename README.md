# eslint-config-vue
Personal preferences for vue eslint configuration

## Installation

```bash
    $ npm install @jardenliu/eslint-config-vue
```

## Usage

add extends `@jardenliu/vue` into your eslint config `.eslintrc`

```json
{
    "root": true,
    "extends": [
        "@jardenliu/vue"
    ],
    "rules": {
        
    }
}
```
